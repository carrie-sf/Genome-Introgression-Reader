## Genome Introgression Reader
<div>
    This contains the files for analyzing a genome in terms of inheritance patterns
<div>
    You may need to replace the separators in the CSV parser commands and also check
    that you have the correct CSV labels. There are many aspects that are specific to
    the files I run but there are examples of these files in the test folder.

<div>
    Code Folder: contains 4 files, each one labled with the general idea of what they do

<div>
    USAGE
<div>
    1. Navigate to the code folder: `cd code`
<div>
    2. Run the main script: `python main.py`
<div>
    3. Use the test CSV files from the test-data folder

<div>
    FEATURES
<div>
    - CSV file parsing
<div>
    - Taking bin coordinates (for previously classified bins)
<div>
    - Using those coordinates and implementing them to parse through each SNP in that bin
